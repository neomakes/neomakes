# On-device AI agents for field operators in network-denied environments.

### Building intelligent systems that:
1. Augment human judgment in extreme environments
2. Operate without cloud dependency

Keywords: on-device AI, edge intelligence, cognitive augmentation
Rooted in: mathematics, physics, cognitive science, defense systems

## 📂 Projects

### 🫡 NeoROGER — Field Decision Support
An on-device agent that filters critical voice communications
and supports real-time decision-making in high-noise,
comms-denied environments.

### 🔬 NeoLAMINAR — Local Intelligence Research
Exploring neural field models and cortical column dynamics
as the foundation for efficient, brain-inspired edge agents.

### 🛠️ NeoLAT — Local Agent Testbed
An automated evaluation framework for local AI systems —
benchmarking hardware efficiency, latency, and agent behavior.
